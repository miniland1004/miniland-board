안내 미니랜드 오류해결

작성자: 미니
안녕하세요 운영진입니다

게임 접속 및 플레이 중 발생하는 주요 오류 현상과 해결 방법을 안내해 드립니다
문제가 발생했을 경우 아래 항목들을 먼저 확인해 주시기 바랍니다

◆ 1. 필수 설치 프로그램 및 운영체제
  • [필수] C++ 2022 RUNTIME 설치
    📥 C++ 2022 x86 다운로드
  • Windows XP는 지원되지 않으며 Windows 10 사용을 권장합니다
  • Windows 7 / Vista의 경우 호환성을 'Windows XP 서비스팩3'으로 설정해 보세요
  • 업데이트 확인을 통해 윈도우를 최신 빌드로 업데이트하세요
◆ 2. 해상도 및 실행 오류 해결
  • 커스텀 해상도 오류 (Failed in finding proper screen mode for Gr2D)
    800x600이 아닌 커스텀 해상도 적용 시 문제가 발생하면 Directx 런타임 및 그래픽카드 드라이버를 최신으로 설치하세요
  • 사용자 지정 해상도 옵션 추가 (AMD / INTEL)
    자신의 그래픽카드 제조사에 맞춰 제어판에서 1024x768 / 1366x768 / 1920x1080 해상도를 수동으로 추가해야 합니다
    🔗 INTEL 해상도 추가 가이드 🔗 AMD 해상도 추가 가이드
  • 검은 창에서 바로 종료되거나 지속적으로 로그인 창으로 가는 경우:
  • 위 현상은 C++ 2022 x86 런타임이 설치되지 않아 발생할 가능성이 높으므로 1번 항목의 필수 프로그램을 꼭 설치해 주세요
◆ 3. 백신 충돌 및 인게임 렉 현상
  • 백신 프로그램 충돌 (바이러스 오진)
    일부 백신에서 넥슨 클라이언트 변조를 감지해 바이러스로 오진하는 경우가 있습니다 백신을 잠시 종료하거나 예외 처리를 진행해 주세요
  • 게임 실행 후 극심한 렉 발생
    그래픽 카드와의 충돌 문제일 확률이 높습니다 그래픽카드 드라이버를 완전히 삭제 후 재설치해 보세요
◆ 4. 키보드 끊김 현상 해결 방법

키보드 입력이 끊기거나 씹히는 경우 아래의 단계별 조치를 진행해 주세요

  • 0단계: 키보드 입력 테스트 및 반응속도 최적화
    ⌨️ 온라인 키보드 테스트 진행 🚀 키보드 반응속도 최적화 가이드
  • 1단계: 윈도우 언어 설정 변경
    설정 ➔ 한국어 ➔ Microsoft 입력기 ➔ 키보드 옵션 ➔ '이전 버전의 Microsoft IME' 켬
    참고 링크 가이드 보기
  • 2단계: 필터키 설정 및 레지스트리 수정
    필터키를 켜고 반복 설정을 진행해야 합니다 (레지스트리 수정 후 재부팅 필수)
    참고 링크 가이드 보기
  • 3단계: 마지막 윈도우 업데이트 제거
    최근 업데이트된 잘못된 내역을 제거한 뒤 윈도우 자동 업데이트를 꺼주세요
    업데이트 제거 가이드 업데이트 끄기 가이드
  • 🎥 영상 가이드 (Windows 10/11)
    ▶️ 유튜브 해결 가이드 시청
안내해 드린 방법으로도 해결되지 않을 경우 디스코드 오류 제보를 이용해 주시기 바랍니다
